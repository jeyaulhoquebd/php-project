# 🔐 Secure PHP & MySQL Login-Registration System

একটি সহজ, সুরক্ষিত এবং ডাইনামিক ইউজার লগইন এবং রেজিস্ট্রেশন সিস্টেম যা **PHP** এবং **MySQL (Procedural/Procedural Method)** ব্যবহার করে তৈরি করা হয়েছে। এই প্রজেক্টটিতে পাসওয়ার্ড সুরক্ষার জন্য আধুনিক হ্যাশিং মেথড এবং সেশন ম্যানেজমেন্ট ব্যবহার করা হয়েছে।

---

## 🚀 বৈশিষ্ট্যসমূহ (Features)

* **ইউজার রেজিস্ট্রেশন:** নতুন ব্যবহারকারীরা ইউজারনেম, ইমেইল এবং পাসওয়ার্ড দিয়ে অ্যাকাউন্ট তৈরি করতে পারবেন।
* **নিরাপদ পাসওয়ার্ড:** ব্যবহারকারীর পাসওয়ার্ড সরাসরি ডাটাবেজে সেভ না করে `password_hash()` দিয়ে এনক্রিপ্ট করে সংরক্ষণ করা হয়।
* **ইউজার ডুপ্লিকেট চেক:** একই ইউজারনেম বা ইমেইল দিয়ে দ্বিতীয়বার অ্যাকাউন্ট খোলা প্রতিরোধ করে।
* **সুরক্ষিত লগইন:** সেশন (`session_start()`) ব্যবহারের মাধ্যমে ইউজার লগইন স্টেট ধরে রাখা হয়।
* **ড্যাশবোর্ড প্রটেকশন:** লগইন ছাড়া কোনো ইউজার সরাসরি ড্যাশবোর্ড পেজে প্রবেশ করতে পারবে না।
* **সহজ লগআউট:** সেশন ধ্বংস (Destroy) করার মাধ্যমে নিরাপদ লগআউটের ব্যবস্থা।

---

## 🛠️ প্রযুক্তি সমূহ (Technologies Used)

* **Backend:** PHP 8.x
* **Database:** MySQL / MariaDB
* **Frontend:** HTML5 (মৌলিক ইন্টারফেস)
* **Server Environment:** XAMPP / WAMP / Live Web Hosting
